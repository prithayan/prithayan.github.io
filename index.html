Prithayan is a PhD student working with Vivek Sarkar at Georgia Tech.
He is interested in program analysis techniques that balance Performance, Portability and Productivity.

Program Transformations like thread coarsening and loop unrolling improve the Performance.

Debugging tools like OMPSan address the Productivity.

DSL compiler optimizations like T2S based on Halide address the Portability.

Optimizations for FPGA

Loop Unrolling

Developing a cost model to select optimal loop unroll factor, implementing in LLVM

Fusion for XLA, Tensorflow

Working on developing a globally optimal kernel fusion algorithm, for a given computation graph.
